# Critique

Use this skill when the user wants a balanced, critical assessment.

## Cover

- **Strengths** — what the paper does well.
- **Weaknesses / limitations** — gaps, unsupported claims, missing baselines.
- **Threats to validity** — confounds, dataset bias, evaluation issues.
- **What would strengthen it** — concrete experiments or analyses.

Be fair and specific; cite the paper's own statements where possible.
