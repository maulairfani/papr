# ELI5 / Feynman explanation

Use this skill when the user wants an intuitive, jargon-free explanation.

## How to explain

1. State the core idea in one plain sentence a curious 12-year-old could follow.
2. Replace every technical term with everyday language; if a term is unavoidable,
   define it with a concrete analogy.
3. Build complexity in small steps, checking understanding as you go.
4. End with "why it matters" in one sentence.

## Avoid

- Equations and notation unless the user asks.
- Restating the abstract verbatim.
