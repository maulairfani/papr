# Analogies

Use this skill when an abstract idea needs grounding in something familiar.

## How

1. Identify the core mechanism to convey.
2. Pick a familiar domain (cooking, traffic, sports) with the same structure.
3. Map each part of the concept to a part of the analogy, explicitly.
4. Note where the analogy breaks down — every analogy has limits.
